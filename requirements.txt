# 1. FFmpeg (REQUIRED for audio processing):
#    - Windows: choco install ffmpeg OR download from https://ffmpeg.org/

#    - Required for some Python packages
#    - Download from: https://aka.ms/vs/17/release/vc_redist.x64.exe
#
# 3. GPU Support (Optional, for CUDA acceleration):
#    - Install PyTorch with CUDA support:
#      pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
#    - Requires NVIDIA GPU with CUDA drivers installed
#
# 4. HuggingFace Hub (for model downloads):
#    - Optional CLI tools: pip install "huggingface-hub[cli,hf_transfer]"
#
# ============================================================================
# INSTALLATION NOTES
# ============================================================================
#
# Complete Installation Steps:
#
# 1. Install system dependencies (ffmpeg, etc.)
# 2. Create virtual environment: python -m venv venv
# 3. Activate: venv\Scripts\activate (Windows) or source venv/bin/activate (Linux/Mac)
# 4. Upgrade pip: python -m pip install --upgrade pip
# 5. Install requirements: pip install -r requirements.txt
# 6. For GPU support: pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
#
# Troubleshooting:
# - If opencv fails: pip install opencv-python-headless
# - If faiss-cpu fails on Windows: pip install faiss-cpu --no-cache-dir
# - If torch CUDA not detected: Reinstall with CUDA-specific wheels
# - If bcrypt fails: pip install bcrypt --no-cache-dir
# - If simpleaudio fails: pip install simpleaudio --no-cache-dir
# - If librosa fails: pip install librosa --no-deps && pip install numba resampy
